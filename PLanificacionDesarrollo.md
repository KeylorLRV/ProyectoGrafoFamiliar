# ProyectoGrafoFamiliar
## Introducción al Proyecto
El proyecto consiste en desarrollar una aplicación de árbol genealógico en C# que modele una familia como un grafo, permita ingresar datos de miembros (foto, cédula, coordenadas de residencia, fecha de nacimiento y edad), visualice un mapa mundial con ubicaciones y fotos, calcule distancias entre familiares al hacer clic en una foto, y genere estadísticas como pares más lejanos/cercanos y distancia promedio. Debe incluir al menos 10 pruebas unitarias, documentación en la wiki de GitHub y una defensa presencial. Dado que el documento original menciona grupos de 3 estudiantes, pero tú indicas 2 personas, adaptaré la planificación para optimizar el trabajo en equipo, dividiendo tareas de manera equilibrada para evitar sobrecargas y fomentar colaboración.
## Planificación del Desarrollo
La planificación se divide en fases secuenciales, con estimaciones de tiempo basadas en un proyecto de 4-6 semanas (ajustable según disponibilidad). Cada fase incluye tareas específicas, asignación para 2 personas (Persona A y Persona B), y hitos. Se asume un ritmo de 10-15 horas semanales por persona. Usa GitHub para control de versiones, con ramas para cada fase y pull requests para revisiones mutuas.
